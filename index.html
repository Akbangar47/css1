What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes the how HTML elements are to be displayed on screen,or in other media.

  3. Css is not neccesary CSS saves a lot of work. It can controls the layout of multile web pages all at once

  4. External stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline-by using the style attribute inside HTML elements
  • Internal-by using a style element in the head section
  • External-by using a link element to link to an external CSS file.
  • The most common ways to add CSS, is to keep the styles in external CSS files.However in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to one unique elements. ID is also special in that you can use a special URL to link directly to an elements and it's used by JavaScript.

In CSS, selectores are used to target a specific element or range of element on a web page.Once an element has been targeted, a style or set of style can be applied to the elements.

There is a wide range of selectors available.Two of the most commonly used class and ID. Both are used to target elements to which a style should be applied.
